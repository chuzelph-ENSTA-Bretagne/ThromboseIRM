    Unsupervised classification

    The algorithm used here is the spectral classification. The main interest of this spectral classification is that it is adapted to signal classification. Thanks to the perfusion IRM, we can extract signal which provide information about the part of the brain we are studying. The whole code can be reused for other application as long as you reshape your input inside a matrix m*n where the row m correspond to the sample m and n correspond to the descriptors of each sample.
