    Welcome to Thrombose IRM Page.

    This page provide the result of my bachelor study I have realize at the ENSTA Bretagne with the help of the CHRU Cavale Blanche of Brest and the INSERM of Lille.

    The hole code of this project is realized in Matlab. The code can be download and the IRM used are aviable as a set of matlab file (.mat).

    Unsupervised classification

    The algorithm used here is the spectral classification. The main interest of this spectral classification is that it is adapted to signal classification. Thanks to the perfusion IRM, we can extract signal which provide information about the part of the brain we are studying.
